# ts-express
> Minimal Express/TypeScript boilerplate

### :wrench: Setup

```bash
cp .env.sample .env # change values after copying
npm i
npm run dev
```

### :construction_worker: Build

```bash
npm run build
npm start
```

### :rotating_light: Testing

```bash
npm test
```

### :arrow_right: Usage

```
GET /
GET /api/custom
```

### :package: Technologies used
* Node.js, TypeScript, Express
